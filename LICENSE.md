## 🔍 Troubleshooting & Common Issues

### The toolkit doesn't load after pasting the snippet

Make sure you're using the latest snippet from the repository. Clear your console cache and try again.

Check that local storage permissions are enabled in your platform application.

### Lag or frame drops during use

Lower the `combat_radius` value and disable unused features. On mobile, close background applications.

### Auto parry misses some attacks

Switch to `aggressive` parry sensitivity. Some enemy special attacks may need an update — check the changelog.

### Fishing module doesn't auto-cast

Ensure you're in a valid fishing zone and that `fish_auto_cast` is set to `true`.

### Mobile overlay doesn't appear

Restart the platform application and re-execute the snippet. On Android, verify overlay permissions in device settings.

### Raid automation stops mid-run

Usually caused by network timeout. Ensure a stable connection and set `raid_auto_retry` to `true`.
